About This Tool

This tool helps you identify potentially malicious websites by checking URLs against:

PhishTank

Database of known phishing sites maintained by the community

VirusTotal

Multi-engine virus and URL scanner with comprehensive analysis

Local Blacklist

Your personal database of manually added suspicious URLs

About VirusTotal API

For full VirusTotal functionality, you need to set a VirusTotal API key as an environment variable. Without an API key, the application will still work but will only use PhishTank and local checks.

Note: No security check is 100% accurate. Always exercise caution when visiting unfamiliar websites.
